# Security Policy

Do not attach real VCF files to public issues or pull requests. VCF feeds may contain cardholder names, employee identifiers, supplier data, account numbers, and transaction history.

If you find a security issue in the parser, please open a private advisory on GitHub or contact the maintainer directly.

Before sharing parser logs or fixtures, remove:

- Card or account numbers.
- Employee IDs.
- Cardholder names and emails.
- Supplier transaction history.
- SFTP hostnames, usernames, keys, and PGP material.
